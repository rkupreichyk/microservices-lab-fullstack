## Laboratory Details

---

# Lab 1: Java SE
Implemented core entity classes with 1:N relationships and DTOs.

Utilized Stream API for data filtering, transformation, and sorting.

Implemented binary serialization and parallel processing using ForkJoinPool.

# Lab 2: Spring Framework & Data
Migrated the project to Spring Boot with Spring Data JPA.

Configured an in-memory H2 database with snake_case naming conventions.

Implemented the Service-Repository pattern and a Command Line Runner for data interaction.

# Lab 3: Spring MVC & REST
Developed REST Controllers (@RestController) with hierarchical resource addresses.

Implemented DTOs for safe data exchange and collection reading.

Documented and tested API endpoints using .http files.

# Lab 4: Microservices & Gateway
Decomposed the monolith into two standalone microservices (Categories and Elements).

Established inter-service event-based communication via REST.

Implemented Spring Cloud Gateway for centralized request routing.

# Lab 5: Angular Framework
Developed a Single Page Application (SPA) using Angular.

Implemented CRUD views for managing categories and elements.

Integrated the frontend with the Gateway microservice to consume backend APIs.

# Lab 6: Docker & Orchestration
Containerized the Angular application using NGINX as a reverse proxy.

Created Docker images for Spring Boot microservices based on Eclipse Temurin.

Orchestrated the entire system (Frontend, Backend, and Databases) using Docker Compose.

---

## Technologies Used
Backend: Java, Spring Boot (Data JPA, MVC, Cloud Gateway), H2/PostgreSQL.

Frontend: Angular, TypeScript, NGINX.

DevOps: Docker, Docker Compose, Maven.
